## Example: Software Engineer

**Must-haves:**
- 3+ years shipping production code
- Experience with our stack (or ability to learn quickly)
- Has built something from scratch to production
- Can discuss tradeoffs in their technical decisions

**Nice-to-haves:**
- Open source contributions
- Technical blog or writing
- Experience at similar stage company
- Domain expertise relevant to our product

**Red flags for this role:**
- Only worked on maintenance, never greenfield
- Can't explain their own code
- No interest in product/users
- Struggles to work independently

## Example: Operations Manager

**Must-haves:**
- Has built processes from scratch
- Tool-fluent (spreadsheets, project management, automation)
- Experience managing cross-functional projects
- Clear written communication

**Nice-to-haves:**
- Startup experience
- Has automated their own workflows
- Financial operations background
- Experience with our industry

**Red flags for this role:**
- Needs constant direction
- Rigid about process over outcomes
- Poor documentation habits
- Can't prioritize independently
